# AIKO 2: Robotaxi - Guida Illustrazioni

## Processo Generazione Immagini

**Illustratrice**: Pina Pennello (agente Onde)
**Tool**: Grok via web (GRATIS con X Premium)

### Procedura Standard

1. **Vai su Grok**: `x.com/i/grok`
2. **Clicca "Create Images"**
3. **Tecnica immagine di riferimento** (OBBLIGATORIA per coerenza):
   - Genera/scegli UN'immagine di riferimento del personaggio principale
   - Carica quell'immagine su Grok (drag & drop)
   - Usa prompt semplici: "stesso bambino, ma [nuova scena]"
4. **Fai SEMPRE upscale** dopo la generazione
5. **Scarica** in `assets/images/chapters/`

### Stile Onde

```
European watercolor style, Beatrix Potter meets modern tech,
soft brushstrokes, warm golden light, natural skin tone,
NO rosy cheeks, NO Pixar, NO 3D, NOT cartoon, 4k
```
